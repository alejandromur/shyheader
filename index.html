Shy Header

This jQuery Plugin shows/hides the header of the page due to its own shy nature.

Once initialized, the header will hide when users scroll the page down more than its height. However, the header remains watching the page so when users scroll the page up its being showed.

Please, take this demo as it is, a little demo where I pretend to show the power of this plugin, but consider your own target before copy/paste. It is possible that you need to reset some properties contained in the CSS file.
